# Runtime Binaries

Coloca aquí los ejecutables y DLLs de inferencia compilados para que la aplicación funcione en modo de desarrollo:

* `llama-diffusion-cli.exe`
* `sd-cli.exe`
* `ggml.dll`, `ggml-base.dll`, `ggml-cpu.dll`, `ggml-cuda.dll`, `llama.dll`, `llama-common.dll`

*Nota: Estos binarios no se suben al control de versiones por su tamaño. Puedes descargarlos de los releases correspondientes del proyecto.*
